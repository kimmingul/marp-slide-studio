# Brief: Gallery Sampler

## Purpose

This is not a real deck. It exists **only** to showcase each theme across all 7 required layout classes, so the gallery can render every theme against the same content.

## Rules

- No brand-specific vocabulary (can't mention "Stripe" or "Apple")
- All copy is in Korean so `lang="ko-KR"` kicks in and typography is tested in the target language
- Content is about slide design itself — meta, generic
- Each slide targets exactly one layout; no mixing

## Beats

| # | Layout | Content focus |
|---|--------|---------------|
| 1 | hero | Theme signature: opening moment, inverse surface |
| 2 | monumental | Single declarative sentence at display scale |
| 3 | split-60-40 | Text + visual asymmetric, typography rhythm |
| 4 | metric | Dominant number + caption, accent usage |
| 5 | divider | Section transition, chapter number |
| 6 | enumerated | 3-item list with hairline rules |
| 7 | quote | Testimonial with attribution |
